- ```html
  <!DOCTYPE html>
  <html>
    <head>
      <meta charset="utf-8">
      <title></title>
    </head>
    <body>
      <script>
        alert("Hello World");
      </script>
    </body>
  </html>

  - case 2:	개발자 도구(**F12**)를 이용한 출력

    3. console 탭 클릭
